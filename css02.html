선택자

스타일을 부여할 대산을 지정하는 키워드

기본 선택자

태그, 클래스, 아이디, 전체 선택자

* { } : 모든 태그
a { } : a 태그만
.클래스명 { }
#아이디명 { }

클래스(class) : 스타일이나 스크립트를 공통으로 적용시키기 위한 것

아이디(id) : 하나의 html 문서에서 각 각의 요소를 중복되지 않게 아이디를 부여하여 각 요소를 구분하여 스타일이나 스크립트를 각각 적용하기 위해 태그에 지정
