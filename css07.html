그림자 속성

shadow : text-shadow, box-shadow

텍스트나 박스의 외부 그림자 또는 내부 그림자를 말하며, 크기 속성에 포함되지 않는다.



텍스트 그림자

text-shadow : x y blur color

x를 양의 값으로 입력하면, 그림자의 방향은 오른쪽에 발생되며, 음의 값으로 입력하면, 왼쪽에 그림자가 발생됨.
y를 양의 값으로 입력하면, 그림자의 방향은 아래쪽에 발생되며, 음의 값으로 입력하면, 위쪽에 발생된다.
x y blur color 양을 여러 단계에 걸쳐 지정하면, 입체모양의 그림자가 표시된다.



박스 그림자

box-shadow : x-length y-length blur-length spread-length color

x를 양의 값으로 입력하면, 그림자의 방향은 오른쪽에 발생되며, 음의 값으로 입력하면, 왼쪽에 그림자가 발생됨.
y를 양의 값으로 입력하면, 그림자의 방향은 아래쪽에 발생되며, 음의 값으로 입력하면, 위쪽에 발생된다.
x y blur color 양을 여러 단계에 걸쳐 지정하면, 입체모양의 그림자가 표시된다.

box-shadow나 text-shadow의 그림자의 양은 px또는 pt 단위로 기재한다.
