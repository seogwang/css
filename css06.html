글자/문단 관련 속성

color, font-size, font-weight

글꼴 지정

font-family : 1차폰트[, 2차폰트][, 3차폰트]

서버에 설치되어 있는 폰트의 이름을 지정하며, 만약 폰트의 이름 중간에 공백이 있거나 아니면, 한글과 같은 영문이 아닌 경우는 반드시 인용부호인 ""나 ''로 감싸서 표기 해야 하며, 만약, 1차 폰트가 존재하지 않는 경우는 2차 폰트가 적용되고, 여기서 2차 폰트도 적용될 수 없다면 3차 폰트가 적용되게 된다. 다만, 3차 폰트가 적용되지 못하는 경우를 대비하여 3차 폰트는 브라우저에 기본적으로 제공하는 폰트를 지정하여야 함.

브라우저 제공하는 기본 폰트는 해당 서버나 클라이언트에 설치가 되어 있지 않아도 자동 제공되는데, sans-serif(고딕계열), serif(명조계열), cursive(궁서계열), monospace(장평이 같은 폰트), fantasy(디자인계열) 등을 말함.

커서 위치의 자간

font-kerning : auto | normal | none



자간

letter-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

해당 요소 안의 글자와 글자 사이의 간격



어간

word-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

단어와 단어 사이의 간격



행 높이

line-height : normal | number | length

값으로 숫자만 기재할 경우는 현재 문단의 글자 크기의 배수가 되며, 숫자 뒤에 단위가 들어가는 경우는 해당하는 단위의 크기가 적용됨



문단 정렬

text-align : left | right | center | justify

문단의 가로 방향에 대한 정렬



들여쓰기/내어쓰기

text-indent : length

length 단위 : px, %, pt 등

들여쓰기 : 다른 줄에 비해 첫 전째 줄이 더 들어가게 쓰는 것인지 text-indent: 20px;

내어쓰기 : 다른 줄에 비해 첫 번째 줄이 더 나오게 쓰는 것 text-indent : -20px;



글자 장식

text-decoration : none | underline | overline | line-through

글자의 위, 중간, 아래에 줄을 표시하도록 하는 속성

none을 제외한 다른 값은 중복해서 사용할 수 있음
text-decoratin:overline underline
text-decoratin:underline double #369, overline wave #c00;

세부속성

text-decoration-color : 줄의 색

text-decoration-line : 줄의 위치(underline/overline/line-through)

text-decoration-style : 줄의 종류(solid/wave/double/dotted/dashed/..)



배분정렬

text-justify : auto | none | inter-caracter | inter-word

단어 또는 글자 단위로 한 줄의 데이터를 분량을 똑같이 배분정렬할 수 있는 속성.



글자 진행 방향

text-orientation : mixed | upright | sideways | sideways-right | use-glyph-orientation



글자 흘러 넘침

text-overflow : clip | ellipsis | 특정문자

단, text-overflow는 overflow가 hidden 이며, white-space: norap인 경우에 유효하다.



내용 보호/줄 끝 처리

white-space : normal | nowrap | pre | pre-line | pre-wrap

하나의 단어가 한 줄에 입력되어야만 하기 때문에 그 때 한 줄에 단어가 배치가 되든지, 아니면, 영문배치문법을 무시 하고 단어의 끝을 어떻게 처리할지 정학는 속성



단어 처리

word-break : normal | break-all | keep-all | break-word

한 줄 또는 한 문장 안에서의 단어의 끝 처리 부분을 어떻게 할지
